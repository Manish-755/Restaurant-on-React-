# Restaurant Management Web App

This is a full-stack Restaurant Management application designed to provide a seamless dining and ordering experience.

## About the Project
I created a complete full-stack web application for a restaurant management system. The project features a dynamic and responsive user interface for customers to browse menus and services, integrated with a secure backend API to handle data operations. I used **React 19** with **Vite** for the frontend to ensure a fast and modern user experience, and **Node.js** with **Express** for the backend to manage server-side logic and database interactions. For data storage, I utilized **MongoDB Atlas**, and implemented **JWT (JSON Web Tokens)** for secure authentication and **Express Rate Limit** for enhanced security.

## Technologies Used

### Frontend
- **React**: Modern library for building user interfaces.
- **Vite**: High-performance development server and build tool.
- **React Router**: For efficient client-side routing.
- **CSS**: Custom styles for a premium design.

### Backend
- **Node.js & Express**: Core framework for building the RESTful API.
- **MongoDB & Mongoose**: NoSQL database for flexible data management.
- **JWT**: Secure authentication system.
- **CORS & Express Rate Limit**: Security middleware for the API.
